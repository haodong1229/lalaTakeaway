分享红包活动玩法：
     1. 分享人给好友发红包，邀请好友成为平台新用户，好友领取红包并完成首单，您即可获得奖励红包
     2. 分享人的好友必须是从未使用过平台的新用户
     3. 好友领取红包后需在平台完成首单(注意：只是首单奖励)，且无退款行为
     4. 奖励红包将会在订单完成的24小时内发送到分享人的账户中
