催单设置

分钟
用户在下单后多少分钟可以开始催单，如设置3分钟，那么用户下单后3分钟之后才可以进行催单操作。不填写为不限制。
分钟
用户在下单后可进行多次催单,该项可设置催单间隔.如:用户现在进行催单,如果设置了10分钟的间隔,那么用户下次催单只能在10分钟以后
